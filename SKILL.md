# Mission-Driven Morphogenesis Theory

Use this skill to turn a theory of artifact form into concrete design decisions.
Treat it as a packaged working form of the source theory supplied by the user.
Treat the theory as a three-way balance between context, readability, and manufacturability.

## Core Rule

Make macro form follow mission and context first.
Make micro structure follow function and operation second.
Never let surface styling override causality or production reality.

## Workflow

1. State the mission in one sentence.
2. Fix the origin, environment, operation, and history of the artifact.
3. List production constraints: material, process, budget, time, medium, body, and safety.
4. Draft the macro silhouette, proportions, massing, and front-back bias.
5. Place micro structures only where function and operation demand them.
6. Add readable cues that communicate use, origin, and condition at a glance.
7. Adjust history so wear, repair, maintenance, or preservation feel earned.

## Evaluation

Check every proposal against these questions:

1. Can the mission be read from the overall form in a few seconds?
2. Can the origin or production culture be inferred from structure and detailing?
3. Can the environment and operation be explained by the layout, access, and handling?
4. Does each major form element have a clear causal reason to exist?
5. Does the design still work when unnecessary detail is removed?
6. Is the result actually buildable in the stated material, process, budget, and medium?

Reject designs that are readable but causally empty.
Reject designs that are causal but unreadable.
Reject designs that are both convincing on paper and impossible to realize.

Use concise output when the user wants a quick judgment.
Use proposal mode when the user needs a design rationale with tradeoffs.
Use checklist mode when the user wants a repeatable studio workflow.
Use critique mode when the user wants pressure-testing against the theory.
